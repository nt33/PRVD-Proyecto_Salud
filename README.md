## Sources

## Download data
1. Download [INE data](https://www.ine.es/jaxiT3/files/t/es/csv_bdsc/68535.csv) into `raw_data/INE-poblacion-grupo-edad.csv` folder.
2. Download [GVA data](https://terramapas.icv.gva.es/0105_Delimitaciones?request=GetFeature&service=WFS&version=2.0.0&typename=ICV.Municipios&outputformat=csv) into `raw_data/GVA-municipios-delimitaciones.csv` folder.
3. Download [Banco de Datos data](https://dadesobertes.gva.es/dataset/0f78e3f9-6137-4ece-aa3f-be951c35de69/resource/34437ccd-288e-41cc-9a95-ea29a01588fd/download/sip_mapa_sanitario.csv) into `raw_data/BancoDeDatos-centros-sanitarios.csv` folder.
